## Visual References

### Imperial / Forbidden City Feeling

**What to search for inspiration:**
- "Forbidden City at dusk photography"
- "Ming Dynasty furniture minimal"
- "Chinese lattice window patterns abstract"
- "Imperial palace shadows photography"
- "Vermillion walls Beijing minimal"

**Key visual elements:**
- Precise geometric lattice patterns
- Deep shadows with warm red undertones
- Bronze/gold metallic accents
- Mathematical precision in architecture
- Mystery through shadow and depth

### Frontier Research / Tech Feeling

**What to search for:**
- "CERN control room photography"
- "Data visualization abstract dark"
- "Cryptographic diagrams aesthetic"
- "Particle physics visualization"
- "Zero knowledge proof diagram"

**Key visual elements:**
- Thin precise lines
- Flowing particle systems
- Blue/purple gradient glows
- Circuit-like patterns
- Clean, minimal compositions

## Midjourney Style Modifiers

Add these to your prompts for consistent style:

**For cinematic quality:**
```
cinematic lighting, 8k, premium quality, moody atmosphere
```

**For dark mysterious feel:**
```
deep shadows, mysterious, ethereal, volumetric lighting
```

**For precision/tech:**
```
precise lines, mathematical, geometric, minimal
```

**For motion-ready frames:**
```
wide shot, negative space, atmospheric depth
```

## Example Prompt Structures

### Pattern 1: Location/Scene
```
[subject], [style reference], [lighting], [mood], [technical] --ar 16:9 --v 6.1
```

Example:
```
abstract geometric lattice pattern, inspired by forbidden city architecture, 
dramatic side lighting, mysterious sacred atmosphere, 8k cinematic --ar 16:9 --v 6.1
```

### Pattern 2: Abstract Visualization
```
[abstract concept], [color palette], [background], [mood], [technical] --ar 16:9 --v 6.1
```

Example:
```
particles of blue purple light flowing through dark space, 
subtle vermillion red undertones, deep black background, 
ethereal mysterious atmosphere, premium quality --ar 16:9 --v 6.1
```

### Pattern 3: Fusion/Transition
```
[element A] morphing into [element B], [color treatment], [mood], [technical] --ar 16:9 --v 6.1
```

Example:
```
traditional Chinese lattice pattern transforming into digital circuit traces, 
gradient from warm bronze to cool blue purple, dark moody atmosphere, 
cinematic visualization --ar 16:9 --v 6.1
```

## What to Avoid

❌ Obvious Chinese motifs (dragons, phoenixes, clouds)
❌ Bright saturated colors
❌ Busy, cluttered compositions
❌ Literal representations
❌ Text or typography in images
❌ Human figures or faces

## What We Want

✓ Abstract geometric patterns
✓ Subtle color with deep blacks
✓ Mysterious atmosphere
✓ Precision and elegance
✓ Lots of negative space
✓ Slow, contemplative mood

---

## For Runway ML Motion Prompts

Keep motion subtle. Example prompts:

**Slow ambient:**
```
gentle ambient movement, particles drifting slowly, breathing light
```

**Camera movement:**
```
slow push in, subtle parallax, cinematic camera drift
```

**Element animation:**
```
pulsing glow, gentle rotation, flowing particles
```

**Avoid:**
```
fast, dynamic, energetic, action, dramatic movement
```

## Final Checklist Before Generating

- [ ] Prompt includes aspect ratio `--ar 16:9`
- [ ] Using Midjourney v6.1 or v6
- [ ] Colors mentioned match our palette
- [ ] Mood is: mysterious, elegant, minimal
- [ ] No obvious literal Chinese imagery
- [ ] Composition has room for logo overlay
